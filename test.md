# Запитання

1. Що таке лінійний алгоритм?
Це алгоритм, який виконує свої дії в певній послідовності, одна за одною, без розгалужень і повторень. 
2. Які оператори вводу/виводу використовуються в C++?
 >> і <<
3. Для чого потрібна директива `#include <iostream>`?
#include <iostream> потрібно для роботи з вводом і виводом даних у мові програмування C++.
4. Чим відрізняється `int` від `double`?
int використовується для зберігання цілих чисел, а double — для чисел з дробовою частиною (дійсних).   
5. Наведіть приклад лінійного алгоритму з життя.
Прання речей у пральній машині:
- Відсортувати білизну за кольором
- Перевірити кишені одягу
- Засипати порошок у відсік
- Покласти речі у барабан
- Закрити дверцята
- Обрати програму прання
- Включити пральну машину
- Дочекатися завершення прання
- Витягнути чисті речі
- Розвісити речі
